This readme file is for the MergeSort and QuickSort algorithms.

Input File:
The input file, named input.txt, must contain a list of doubles separated by spaces. You may include as many doubles as you wish, but every number must be a double.

Output File:
The output file, named output.txt, will contain a sorted list of the inputs in input.txt. This happens after you compile and run either MergeSort.java or QuickSort.java.

Compilation:
To compile MergeSort, type 'javac MergeSort.java'
To compile QuickSort, type 'javac QuickSort.java'

Running:
To run MergeSort, type 'java MergeSort.java'
To run QuickSort, type 'java QuickSort.java'
