This readme file is for Horspool's String Matching algorithm.

Input File:
The first line of the input file (named input.txt) should contain the pattern you wish to search for.
The second line of the input file should contain the string of text you wish to search through. It may include any of the basic alphabetic letters, as well as spaces.

Output File:
The output file (named output.txt) will contain the index of the left end of the first matching substring found.
If no matching substring is found, then the output file will contain -1.

Compilation:
To compile HorspoolStringMatching, type 'javac HorspoolStringMatching.java'

Running:
To run HorspoolStringMatching, type 'java HorspoolStringMatching.java'
